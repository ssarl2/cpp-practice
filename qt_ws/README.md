# Qt library practice(pronounced as "cute")

## Dependencies
`sudo apt-get install qtcreator qt5-default`
```
git clone https://github.com/kisli/vmime.git
sudo apt-get install
\ libgsasl7-dev
\ doxygen
\ graphviz
\ sendmail
```

## Build
```
qt_ws$ mkdir build
qt_ws$ cd build
qt_ws/build$ cmake ..
qt_ws/build$ cmake --build .
```

## Run
```
qt_ws/build$ ./qt
```
